Understanding JavaScript Classes

Welcome to our JavaScript class review!

Explore the concepts through code examples.

Learning Goals:

  • Understand that classes serve as blueprints for creating objects.
  • Learn how to define and instantiate classes.
  • Explore class constructors, properties, and methods.
  • Practice creating multiple instances from a class to see how each instance maintains its own state.
  • Observe how methods can interact with an object's properties and alter its state.

Summary:

In the accompanying JavaScript files, we define a class named MyAwesomeClass with properties like name, flavor, and age, and methods to interact with these properties. We then create multiple instances of this class in script.js, demonstrating the use of class constructors, methods, and the encapsulation of state within objects.

There are 5 parts of a class declaration

  1. Names your class class MyNewClassName {
    • note the opening curly brace
  2. Make the constructor constructor(firstParam, secondParam)
  3. Define Properties {this.myProp1 = 43; this.myProp2 = "potato";}
  4. Define methods myFunc(param) {code and more code}
  5. Close the first curly brace: }

What to Look for in the Console:

  • Instances of MyAwesomeClass being created with different initial values.
  • Use of methods to display or change properties of these instances.
  • The effect of method calls on the state of each object, visible through console output.

Check your browser's developer console to see the output from the JavaScript files and witness the dynamic nature of object-oriented programming in action!
