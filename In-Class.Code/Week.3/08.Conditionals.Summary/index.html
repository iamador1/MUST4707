JavaScript Basics: Introduction to Conditionals and Operators

Comparison Operators

Comparison operators are used to compare values and include == (equal to), > (greater than), < (less than), >= (greater than or equal to), and <= (less than or equal to).

Logical Operators

Logical operators are used to determine the logic between variables or values and include && (and), || (or), and ! (not).

Using Conditionals with Operators

We can use these operators within if, else if, else, and switch statements to make more complex decisions.

Using else if

The else if statement can be used to create multiple conditional branches in your code. It is placed after an if statement and before an else statement. This allows you to specify a new condition to test, if the first condition is false.
