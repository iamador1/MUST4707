JavaScript Basics: User Input, Alerts, and Data Conversion

Getting User Input with Prompt

The prompt() function is used to display a dialog box for user input. This input is always received as a string, even if the user enters numbers.

Converting String to Numbers

When you need to work with numbers (for calculations, comparisons, etc.), you should convert the input string to a number. Use parseInt() for whole numbers and parseFloat() for decimal numbers.

Example: let userAge = parseInt(prompt("Enter your age:")); converts the input age to a number.

Displaying Alerts

The alert() function displays an alert dialog with a specified message. It's a simple way to show information to the user.

Check the console for demonstrations and examples.
